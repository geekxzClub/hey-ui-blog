## 简介

  HBlog 是基于Hey UI原生语法,实现的简单的个人博客,面向全层次的小程序开发者,低门槛开箱即用的个人博客解决方案.

#### 体验地址
  后端源码地址: https://github.com/geekxz/heyui-Hblog-service.git

  pc前端地址：待开发
  
  微信小程序：https://github.com/geekxzClub/heyui-blog.git

#### 主要功能
  - 注册登录：微信小程序授权登录，小程序端可绑定pc端的账号，实现数据同步。
  - 文章管理：文章的详细信息，文章的评论等等。
  - 分类管理：文章的分类管理，分类查找需要的文章。
  - 工具管理：小程序端可查看工具的功能，一键复制工具地址。
  - 用户评论：小程序端只可以查看用户的评论，pc端用户可以评论，同时能过同步到小程序。

## 快速上手

### 安装

  直接通过git下载 HBlog 源代码，然后直接小程序开发者工具导入项目即可.
  ```
  git clone https://github.com/geekxzClub/hey-ui.git
  ```
